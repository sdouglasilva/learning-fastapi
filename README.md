### -Iniciar o gerenciador:
```
uv init
```
### -Instalar o framework necessário:
```
uv add "fastapi[standard]"
```

### -Iniciar o projeto em modo desenvolvedor e o arquivo raiz do projeto:
```
uv run fastapi dev main,py
```
